//放款说明
执行方式：定时任务
脚本路径：php yii "borrow/batchlendpay"
执行时间：每天7点到22:00 每半个小时执行一次
放款流程说明：
1、项目是已经登记过的
2、得到需要放款的项目(登记、满标、未放款、金额匹配)
3、判断资金以及是否提交过放款申请
4、满足条件，把投资债权拼接发送批次放款请求
5、等到银行处理

==>银行返回结果处理
1、更新项目的状态为放款
2、给借款人写资金流水(要判断资金流水是否可写，避免重复)
3、处理每一个投资人的资金冻结问题
4、处理投资人投资红包
5、处理投资人上线提成
6、生成还款计划
