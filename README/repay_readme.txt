//还款说明
执行方式：定时任务
脚本路径：php yii "borrow/batchrepay"
执行时间：每20分钟执行一次 建议早上6点以后执行
还款流程说明：
1、得到截止今天23:59:59为止所有的还款计划
2、判断项目的还款金额合规性
3、判断是新旧项目的还款
->旧项目-用户未开通存管
1、执行以往的还款逻辑
2、还本金、利息、加息券
3、处理投资记录、还款计划、项目详细
->旧项目-用户已经开通存管
1、执行以往的还款逻辑
2、还本金、利息、加息券  以大麦红包的形式反馈到用户账户中
3、处理投资记录、还款计划、项目明细

->新项目
1、判断每一个还款计划资金的合法性
2、拼接批次还款信息，判断是否已经提交请求的重复性
3、批次提交还款请求


==>银行返回结果处理
1、还款投资人资金流水的变更
2、(要判断资金流水是否可写，避免重复)
3、处理还款中大麦加息的部分、处理还款中加息券的部分
4、处理还款计划、投资计划、项目还款信息
5、如果还款完成，需要更改状态，把项目置为可以结束债权的操作

查询：
执行脚本： php yii "borrow/batchrepayquery"
执行时间：8.30-23 每2个小时执行一次
测试可以半小时一次
